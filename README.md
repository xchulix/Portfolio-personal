# Portfolio Personal — sixseven

Este es mi portfolio profesional, diseñado para mostrar mis proyectos, habilidades técnicas y experiencia en el ámbito del desarrollo web, administración de sistemas y tecnologías IT.  
El objetivo es presentar de forma clara y visual mi perfil como candidato para puestos junior en soporte, sistemas, desarrollo y entornos Linux.

🌐 **Demo online:** https://xchulix.github.io/Portfolio-personal/

---

## 🚀 Características principales

- Diseño moderno, minimalista y responsive  
- Secciones claras: presentación, habilidades, proyectos y contacto  
- Navegación rápida y estructura intuitiva  
- Código limpio y organizado  
- Optimizado para GitHub Pages  

---

## 🧩 Tecnologías utilizadas

- **HTML5** — estructura del sitio  
- **CSS3** — estilos, diseño y animaciones  
- **JavaScript** — interacción y comportamiento  
- **GitHub Pages** — despliegue del portfolio
